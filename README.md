# Simple Raffle

A minimal raffle application built with [Stacks](https://www.stacks.co/).

## Features

- Enter a raffle by submitting your entry
- Random winner selection
- Transparent and auditable process

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/simple-raffle.git
   cd simple-raffle
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Run the application:**
   ```sh
   npm start
   ```

## Project Structure

- `src/` — Source code for the raffle logic
- `settings/` — Configuration files (Mainnet.toml, Testnet.toml)
- `.cache/` — Build cache (ignored)
- `logs/` — Application logs (ignored)
- `coverage/` — Test coverage reports (ignored)

## .gitignore

Sensitive and generated files are excluded from version control:
- Network settings
- Cache and logs
- Coverage reports
- Node modules
